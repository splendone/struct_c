# 线性表顺序存储结构动画演示

这个项目包含一个Web页面，用于演示数据结构中线性表顺序存储结构的插入、删除操作的C语言代码运行时的内存变化动画。

## 文件说明

- `linear_list_animation.html` - 主要的动画演示页面，包含完整的HTML、CSS和JavaScript代码

## 功能特点

- 可视化展示线性表的内存布局
- 动画演示插入和删除操作过程
- 显示对应的C语言代码实现
- 提供手动操作和自动演示功能
- 颜色编码帮助理解不同操作状态

## 使用方法

1. 打开 `linear_list_animation.html` 文件
2. 使用浏览器直接打开该文件，或通过HTTP服务器访问
3. 尝试手动插入/删除元素或使用演示功能

## 演示功能

- 插入元素：在指定位置插入新元素，展示元素移动过程
- 删除元素：删除指定位置的元素，展示元素移动过程
- 自动演示：自动展示插入和删除操作
